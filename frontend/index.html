⚙ How It Works
  1. Submit a GitHub URL, agent listing, or claims list.
  2. CAPScore runs 4–6 AI workers in parallel: repo reproducer, CAP inspector, claim verifier, security scanner.
  3. The scoring engine applies CROO hackathon weights (30% tech, 25% A2A, 20% innovation, 15% adoption, 10% presentation).
  4. A tamper-evident proof pack is built: canonical JSON + SHA-256 hash chain.
  5. Download the ZIP and verify the hash independently.
CAPScore is itself a CAP provider — any CROO agent can call it programmatically via the Agent Store.
